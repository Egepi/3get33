void MainWindow::keyPressEvent ( QKeyEvent *  key  ) 

When ever a key is pressed this method is called to decide what action to take.

This method is used for key press events for moving the ship, shooting bombs, shooting guns, and pausing the game. All other keys just do nothing.

void MainWindow::keyReleaseEvent ( QKeyEvent *  key  ) 

When ever a key has been released this method decides to stop the current action.

This method just ends all of the actions that are initiated in MainWindow.keyPressEvent().

void MainWindow::loadGame (  )  [slot]

Prompts user to load a level file using OS generic load file ui.

Slot is called when the 'Load Game' button is clicked from MainWindow.

The generic OS file load window is opened and the current directory from which the game is running is the default folder it opens up to.
The user loads a level file and the game will then run with the level specifications.
